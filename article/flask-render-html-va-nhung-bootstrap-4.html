[FLASK] Render HTML và nhúng Bootstrap 4

Ở bài này mình sẽ hướng dẫn render HTML thay vì render string như bài trước và nhúng thư viện Bootstrap 4.

#python #flask #html #css #js #jquery #bootstrap
by letrii · 05/12/2019

Phần trước Xây dựng website chỉ cần 6 dòng với Python chúng ta đã tìm hiểu cách xây dựng website nhưng chỉ dừng lại ở mức trả về cho người dùng chuỗi Hello, World!. Ở phần này, mình sẽ hướng dẫn trả về HTML và nhúng thư viện Bootstrap 4 để làm cho website đẹp hơn. Để có thể hiểu được phần này, bạn nên có kiến thức về HTML.

Code

Mở file hello.py và chỉnh sử theo nội dung bên dưới:

from flask import Flask, render_template
app = Flask(__name__)

@app.route('/')
def index():
    return render_template('index.html')

Tạo thư mục templates

mkdir templates
cd templates

Tạo tiếp file index.html với nội dung sau đây:

<!DOCTYPE html>
<html>
    <head>
        <meta charset="UTF-8">
        <title>[FLASK] Render HTML và nhúng Bootstrap 4</title>
    </head>

    <body>
        <h2>Hello, World!</h2>
    </body>
</html>

Lúc này cấu trúc thư mục sẽ như sau:

├── __pycache__
├── venv
├── hello.py
└── templates
    └── index.html

Chạy ứng dụng:

flask run

Kết quả:

Nếu bạn thấy chữ Hello, World! to và đậm hơn trước thì đã render HTML thành công rồi đó. Tiếp theo sẽ nhúng Bootstrap 4 vào, mở file index.html và thay đổi nội dung:

<!doctype html>
<html lang="en">
    <head>
        <!-- Required meta tags -->
        <meta charset="utf-8">
        <meta name="viewport" content="width=device-width, initial-scale=1 shrink-to-fit=no">

        <!-- Bootstrap CSS -->
        <link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/4.0.0/css/bootstrap.min.css" integrity="sha384-Gn5384xqQ1aoWXA+058RXPxPg6fy4IWvTNh0E263XmFcJlSAwiGgFAW/dAiS6JXm" crossorigin="anonymous">

        <title>[FLASK] Render HTML và nhúng Bootstrap 4</title>
    </head>
    <body>
        <h2>Hello, World!</h2>

        <!-- Optional JavaScript -->
        <!-- jQuery first, then Popper.js, then Bootstrap JS -->
        <script src="https://code.jquery.com/jquery-3.2.1.slim.min.js" integrity="sha384-KJ3o2DKtIkvYIK3UENzmM7KCkRr/rE9/Qpg6aAZGJwFDMVNA/GpGFF93hXpG5KkN" crossorigin="anonymous"></script>
        <script src="https://cdnjs.cloudflare.com/ajax/libs/popper.js/1.12.9/umd/popper.min.js" integrity="sha384-ApNbgh9B+Y1QKtv3Rn7W3mgPxhU9K/ScQsAP7hUibX39j7fakFPskvXusvfa0b4Q" crossorigin="anonymous"></script>
        <script src="https://maxcdn.bootstrapcdn.com/bootstrap/4.0.0/js/bootstrap.min.js" integrity="sha384-JZR6Spejh4U02d8jOt6vLEHfe/JQGiRRSQQxSfFWpi1MquVdAyjUar5+76PVCmYl" crossorigin="anonymous"></script>
    </body>
</html>

Tắt (CTRL+C) và chạy ứng dụng:

flask run

Lúc này chữ Hello, World! đã khác với ở trên. Bạn có để ý là mỗi lần mình thay đổi code là phải tắt và chạy lại ứng dụng không? Để không phải tắt lại bạn chỉ cần gõ (nhớ tắt ứng dụng):
