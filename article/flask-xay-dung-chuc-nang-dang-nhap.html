Kết quả:

Thay đổi dòng đầu tiên của file app.py thành:

from flask import Flask, jsonify, request, render_template, url_for, redirect

OK, bây giờ đã có giao diện đẹp rồi tiếp theo sẽ viết phần xử lí đăng nhập. Mở file app.py và thay đổi nội dung ở method login:

@app.route('/login', methods=['GET', 'POST'])
def login():
    USER_DB = {
        'username': 'admin',
        'password': 'admin'
    }
    form_data = request.form
    if request.method == 'POST' and form_data:
        username = form_data.get('username', None)
        password = form_data.get('password', None)

        if username == USER_DB['username'] and password == USER_DB['password']:
            return render_template('dashboard.html')

    return redirect(url_for('index'))

Giải thích code:

  • Method login: khi người dùng nhập đúng username và password thì chuyển trang về dashboard.html còn không thì chuyển về index.html. Ở đây này mình chỉ sử dụng biến USER_DB để lưu thông tin người dùng, ở bài sau mình sẽ hướng dẫn sử dụng cơ sở dữ liệu (CSDL - database) để lưu thông tin người dùng.
