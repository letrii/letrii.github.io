Xây dựng website chỉ cần 6 dòng với Python

Xây dựng website chỉ với 6 dòng với Python và Flask.

#python #flask
by letrii · 11/12/2019

Flask là một micro web framework được biết bằng python. Mục tiêu của bài này là ai cũng có thể tự xây dựng một website cho riêng mình (tất nhiên để làm được thì bạn cần phải biết python căn bản)...

Tạo thư mục để quản lý

mkdir myweb
cd myweb

Tạo môi trường (virtualenv)

python3 -m venv venv

Kích hoạt môi trường (virtualenv)

source venv/bin/activate

Cài đặt

pip install Flask

Code

Đầu tiên tạo một file hello.py với nội dung bên dưới:

from flask import Flask
app = Flask(__name__)

@app.route('/')
def hello_world():
    return 'Hello, World!'

Giải thích code trên:

  • Dòng đầu tiên và thứ hai: import thư viện flask và khởi tạo instance cho ứng dụng
from flask import Flask
app = Flask(__name__)
  • Dòng thứ ba: xuống dòng syntax của python
  • Dòng thứ tư và năm: hiểu đơn giản là khi ta vào trang chủ thì sẽ gọi đến hàm hello_world và trả về kết quả là chuỗi Hello, World!. Trong các bài sau mình sẽ hướng dẫn kỹ hơn về route và trả về dạng JSON, HTML.
@app.route('/')
def hello_world():
    return 'Hello, World!'

Chạy thử ứng dụng:

export FLASK_APP=hello.py
flask run

Kết quả:

   WARNING: This is a development server. Do not use it in a production deployment.
   Use a production WSGI server instead.
 * Debug mode: off
 * Running on http://127.0.0.1:5000/ (Press CTRL+C to quit)

Rồi bạn mở trình duyệt của mình lên và gõ http://127.0.0.1:5000/, xem kết quả giống như hình dưới không nhé.
